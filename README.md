# Portfolio - Lisa Anton

Portfolio personnel présentant mes compétences, projets et expériences en tant qu'étudiante en informatique.

## Installation et lancement

### Prérequis

Aucune dépendance externe n'est nécessaire.  Il vous suffit d'avoir un navigateur web moderne.

### Méthode 1 : Ouvrir directement le fichier

1. **Cloner le repository** :
   ```bash
   git clone https://github.com/lyysax/Portfolio.git
   cd Portfolio
   ```

2. **Ouvrir le fichier** :
   - Double-cliquez sur `index.html`
   - Ou faites un clic droit → "Ouvrir avec" → Votre navigateur préféré
